== Installing drivers on Steam Deck

Steam Support:
https://help.steampowered.com/de/faqs/view/6121-ECCD-D643-BAA8

=== GPU

[source,pwsh]
----
GPU\setup.exe
----

=== SD card

Create `sd_card_setup.iss`:

[source,pwsh]
----
Sdcard\setup.exe /r
----

Install driver:

[source,pwsh]
----
Sdcard\setup.exe /s
----
